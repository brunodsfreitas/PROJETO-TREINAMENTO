[UBUNTU 20.04] 
    [REBOOT]
        reboot
    [NETWORK CONFIGURATION]

        Há um gerenciador de interface chamado cloudinit {ESTUDAR DEPOIS, serve para automação} https://www.youtube.com/watch?v=exeuvgPxd-E;
        Precisa ser verificado, através do arquivo:
        '/etc/cloud/cloud.cfg.d/subiquity-disable-cloudinit-networking.cfg'
        Se o conteúdo conter 'network: {config: disabled}', ele não é gerenciado pelo Cloud Init;
        Já que não é, o arquivo de configuração é o: '/etc/netplan/00-installer-config.yaml'
        Abaixo está o exemplo para IP Fixo, deve ser respeitada a identação, DNS incluso:
        {
        network:
            version: 2
            ethernets:
                ens33:
                    addresses: [192.168.20.160/24]
                    gateway4: 192.168.20.2
                    nameservers:
                        addresses: [192.168.20.2, 8.8.8.8]
        }

        O comando para validar a sintaxe do arquivo é: 'netplan try' (Isso já altera o IP)
        Para aplicar é o comando: 'netplan apply'

    [SHOW ROUTE]
        ip route
        netstat -rn

[DEBIAN 11.3]
    [REBOOT]
    /sbin/init 6
    [NETWORK CONFIGURATION]
        systemctl status networking --CHECK NETWORKING
        ls /sys/class/net/ --LIST INTERFACES

        Abaixo está o exemplo de configuração de IP FIXO do arquivo '/etc/network/interfaces':
        {
            auto eth0
            iface eth0 inet static
                address 192.0.2.7/24
                gateway 192.0.2.254
        }

        Para DNS, é através do arquivo '/etc/resolv.conf' (validar se não é link simbólico, se for, é gerenciado por um programa):
        {
            nameserver 12.34.56.78
            nameserver 12.34.56.79
        }

        Desativar IPV6, arquivo '/etc/sysctl.conf':
        {
            net.ipv6.conf.all.disable_ipv6 = 1
            net.ipv6.conf.default.disable_ipv6 = 1
            net.ipv6.conf.lo.disable_ipv6 = 1
            net.ipv6.conf.tun0.disable_ipv6 = 1
        }
